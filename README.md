## Assignment Description
This repository contains the work for HW-06B, which focuses on testing and debugging an existing triangle classification program.

The objectives of this assignment are:
1. Develop a set of test cases for an existing triangle classification program
2. Use those test cases to identify defects in the original implementation
3. Fix the defects in the program
4. Report the testing results for both the original and corrected implementations

## Files Included
- `Triangle.py`  
  Contains the triangle classification function `classifyTriangle()`

- `TestTriangle.py`  
  Contains the unit test cases for testing `classifyTriangle()`

- `initial_test_output.txt`  
  Output from running the expanded test suite against the original buggy implementation

- `final_test_output.txt`  
  Output from running the expanded test suite against the corrected implementation
